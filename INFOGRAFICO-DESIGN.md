## 💡 DICAS DE DESIGN PRO

### **Hierarquia Visual**
```
1. Tamanho: Título > Subtítulo > Corpo
2. Cor: Alto contraste para importante
3. Peso: Bold para destaque, Regular para corpo
4. Espaçamento: Mais espaço = mais importância
```

### **Legibilidade**
```
✅ Contraste mínimo 4.5:1 (WCAG AA)
✅ Tamanho de fonte mínimo 20pt
✅ Line-height 1.4-1.6
✅ Largura de texto máxima 60-80 caracteres
✅ Evitar texto sobre imagens sem overlay
```

### **Composição**
```
✅ Regra dos terços
✅ Alinhamento consistente
✅ Espaçamento uniforme (8px grid)
✅ Balanço visual (peso distribuído)
✅ Ponto focal claro (o que ver primeiro?)
```

### **Cores**
```
✅ Máximo 3-5 cores principais
✅ Use a roda de cores (complementares, análogas)
✅ 60-30-10 rule (60% dominante, 30% secundária, 10% accent)
✅ Teste em modo escuro e claro
```
